Square knot

Boy Scout's inspiration ;)

Knot's are always difficult to fabricate, so i think design one, and then print it.

First i learn with Luis some Rhino. :)

First i use curve command to chose the curve tool to design the knot.

After that, i use "Points on" so i can control all the point's of the curve.

I controle them using Gumball tool, for a more accurate manipulation

After i design the shape as a part of the rope of a knot, i scale it to fit the size i have imagined (cm's).

Than copying that shape and rotate them to fit like a knot in one another

Again i use gumball to adjust the curve point's

PIPE is the magical command that put a pipe along the curve.

So know the knot look's more real and i can print it :)

For use 3D file viewer from github you must remove "blob" word and make it like this.
